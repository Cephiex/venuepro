# VenuePro

**The All-In-One Management Software for Your Building**

From room bookings to maintenance tickets, VenuePro provides a simple, powerful dashboard to manage your association, club, or building's daily operations.

> No credit card required.

---

## Overview

Stop juggling spreadsheets and emails. VenuePro centralizes your building's management in one powerful, easy-to-use platform — giving administrators and members a single place to handle bookings, organizations, maintenance, and access control.

## Features

### 📅 Event & Room Booking
A centralized calendar to manage all room bookings, events, and availability with ease.

### 🏢 Organization Management
Manage member organizations, assign users, and control room access permissions.

### 🔧 Maintenance Ticketing
Streamline repair requests with a simple ticketing system for all maintenance issues.

### 🔐 Role-Based Access
Secure your dashboard with administrator and user roles for controlled access.

## See VenuePro in Action

Each view is designed for clarity and ease of use, helping you manage your building more effectively.

- **Centralized Dashboard** — Get a high-level overview of all building activities, including recent bookings, pending approvals, and open maintenance tickets.
- **Intuitive Calendar** — View and manage all events in a clean, interactive calendar. Quickly see availability and schedule new bookings with ease.
- **Room & User Management** — Easily configure rooms, set permissions for different organizations, and manage all user accounts from one place.
- **Streamlined Repairs** — Keep track of all maintenance requests through a simple and effective ticketing system, from submission to completion.

## Usage

1. Sign in as an administrator to configure rooms, organizations, and user roles.
2. Invite members and assign them to their respective organizations.
3. Members can browse availability and book rooms through the calendar.
4. Submit and track maintenance tickets from request to completion.

## Roles

| Role          | Capabilities                                                        |
| ------------- | ------------------------------------------------------------------- |
| Administrator | Manage rooms, organizations, users, permissions, and all tickets.   |
| User          | Book rooms, view the calendar, and submit maintenance requests.     |
